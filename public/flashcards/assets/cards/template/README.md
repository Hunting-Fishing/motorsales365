# Assets — template

Put card images for the **template** section here.

- Replace `placeholder-card.svg` or add your own files (PNG/JPG/SVG/WebP).
- Reference an image from a card via its `media` field, e.g.:
  `"media": { "image": "../assets/cards/template/my-image.png" }`
- Recommended card art size: 480×300 (or any 8:5 ratio); keep files web-light.
