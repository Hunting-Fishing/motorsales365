# Small Engines → Exhaust & Emissions

Place Small Engines card art for the **Exhaust & Emissions** system here.

Name files by card id:
- `<card-id>-front.png` and `<card-id>-back.png`  (recommended)
- or a single combined `<card-id>.png` (front left / back right)

`png · jpg · webp · svg` all work. Run `node tools/build-index.js` after adding.
