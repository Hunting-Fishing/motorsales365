# Brand assets

Drop the official 365 Motor Sales logo here so it appears in the app header.

- **logo-365.png** — preferred. A square/round transparent PNG (the round
  badge logo works perfectly). Add it here and it is used automatically.
- **logo-365.svg** — the current branded placeholder. Replace it with a vector
  version if you have one.

The header uses `logo-365.png` if present, otherwise `logo-365.svg`, otherwise a
plain "365" text badge. No code changes needed — just drop the file in.
